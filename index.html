Open Ephys GUI¶


This is the new documentation site for the Open Ephys GUI, an open-source, plugin-based application for acquiring extracellular electrophysiology data.

The GUI was designed by neuroscientists to make their experiments more flexible and enjoyable. It works equally well on macOS, Linux, and Windows, although some plugins are only available for certain operating systems.

Previously, the GUI’s documentation lived on the Open Ephys wiki. We are planning to migrate all of the docs to this site over time.
