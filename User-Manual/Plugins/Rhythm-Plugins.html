Rhythm Plugins#

Documentation for plugins that use Intan’s Rhythm API now live on separate pages:

  • OE FPGA Acquisition Board - acquires data from recently purchased Acquisition Boards with a custom FPGA designed by Open Ephys

  • Acquisition Board - acquires data from Acquisition Boards with a Opal Kelly FPGAs

  • Intan RHD USB - acquires data from the Intan RHD USB evaluation board

  • RHD Rec Controller - acquires data from Intan’s 512 or 1024 channel RHD Recording Controllers
